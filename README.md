# rsa-haskell for functional programming classes in AGH UST



### Quick start
Requirements: Cabal, Stack +++++

```bash
git clone https://github.com/gjasinski/rsa-haskell.git
cd rsa-haskell
stack build
stack exec tutaj dopisac
```

####Geneating RSA keys (public and private)
```bash
blebele
```

#### Launching tests:
```bash
stack test
```
